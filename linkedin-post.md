20만원짜리 도메인을 사서 무료 서브도메인 서비스를 만들었습니다.
9개월이 지난 지금, 3,600개 이상의 서브도메인이 등록되었어요.

is-an.ai는 [name].is-an.ai 형태의 서브도메인을 무료로 제공하는 서비스입니다.
사이드프로젝트에 .vercel.app 대신 기억에 남는 도메인을 붙이고 싶을 때, 가볍게 가져가실 수 있어요.

9개월간 정말 많은 일이 있었는데요,

- GeekNews에 홍보한 지 3시간 만에 메인 노출
- 중국 커뮤니티 NodeLoc에서 입소문이 나며 트래픽 폭발
- Cloudflare의 3,500개 레코드 제한에 부딪혀 PowerDNS로 자체 네임서버 구축
- 인도의 고등학생 개발자가 디스코드로 연락해 팀에 합류

그리고 최근에는 CLI와 Claude Code Skill을 만들었습니다.
AI 에이전트가 회원가입 없이 GitHub PR 하나로 도메인을 발급받을 수 있어요.

/plugin install is-an-ai/cli
